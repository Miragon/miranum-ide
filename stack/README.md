# Stack

```bash
docker compose up -d
```

1. Run the docker compose stack `docker compose up -d`
2. Start the backend (digiwf-engine-service) with the profiles `local, no-security, no-mail, streaming, no-ldap`.
3. Go to [localhost:8081](http://localhost:8081)
